3. Rate Limits & Fair Usage
To ensure all teams have equal access to compute resources, please adhere to the following:
● Mandatory Rate Limiting: You must implement client-side rate limiting in your application
code. Do not exceed 10 requests per minute (RPM) for GPT-5.4 and 3 requests per
minute for GPT-image-1.5.
● Moderate Consumption: Use the API only for essential testing and your final demo. Avoid
"looping" prompts or stress-testing the models.
● Monitoring: Nexalaris Tech monitors real-time usage. Excessive or "runaway" requests
may result in temporary key suspension.
4. Security & Best Practices
● Key Safety: Never hardcode your API keys in frontend code or public GitHub repositories.
Use environment variables.
● Data Privacy: Avoid sending sensitive personal data through the endpoints.
● Error Handling: Implement try-catch blocks to handle potential 429 (Too Many Requests)
or 500 (Server Error) responses gracefully in your UI.
5. Endpoint Expiration
● Termination Date: All API keys and access to the AI Foundry endpoints will be
deactivated on March 17, 2026.
● Post-Event Access: Ensure you have recorded any necessary logs or outputs for your
final presentations before this date, as no extensions will be provided.
